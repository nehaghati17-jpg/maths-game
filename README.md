# Maths Game

Maths Game is a browser-based interactive web application built using **HTML, CSS and JavaScript**.
The game generates random multiplication questions with multiple-choice answers.

## Features
- Random maths question generation
- Multiple-choice answer selection
- Countdown timer
- Real-time score tracking
- Instant feedback for correct and incorrect answers

## Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)

## Project Purpose
This project was developed to practice core JavaScript concepts such as event handling,
DOM manipulation and basic game logic while creating a fun and interactive learning experience.
